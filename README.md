# GAZPROM_1
"оцифровка паспортов и создание шкафов"
1. Приложение разработано для работы на python 3.12.5 
2. Для запуска приложения необходимо создать виртуальное окружение .venv
3. Далее установить библиотеки из файла requirements.txt командой pip install -r requirements.txt Для более качественной и быстрой работы приложения необходимо устанавливать библиотеки под видеокарту, в этом случае необходимо удалить из файла requirements.txt библиотеку llama-cpp-python и заменить paddlepaddle на paddlepaddle-gpu). При удалении llama-cpp-python в консоли необходимо ввести следующую команду CMAKE_ARGS="-DGGML_CUDA=on" pip install llama-cpp-python. ДАННОЕ ДЕЙСТВИЕ ПРОВОДИТЬ ТОЛЬКО ПРИ НАЛИЧИИ ВИДЕОКАРТЫ с cuda-ядрами.
5. В папку с файлами программы необходимо поместить файл qwen2.5-3b-instruct-q4_k_m.gguf (можете скачать с гугл диска отдельно)
6. Запустить приложение через файл app.py
7. Принцип работы приложения отображен в видеоматериалах и в рукводстве 

АЛТЕРНАТИВНО
В папке внутри диска представлен zip-архив, можно распаковать его и пользоваться приложением.
