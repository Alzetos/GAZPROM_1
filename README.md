# GAZPROM_1
"оцифровка паспортов и создание шкафов"

1. Приложение разработано и протестировано на версии Python 3.12.5.
2. Для стабильной работы системы необходимо создать виртуальное окружение .venv (делается в случае, если вы не использовали zip-архив). 
3. После активации окружения необходимо установить все зависимости из файла requirements.txt с помощью стандартного менеджера пакетов pip install -r requirements.txt.
4.  Для значительного ускорения работы ПРИ НАЛИЧИИ видеокарты NVIDIA с поддержкой CUDA рекомендуется использовать специализированные версии библиотек. В этом случае необходимо заменить стандартный пакет paddlepaddle на версию paddlepaddle-gpu, а библиотеку llama-cpp-python переустановить с флагом поддержки CUDA (DGGML_CUDA=on). Эти действия следует проводить только при наличии соответствующего графического оборудования.
5. В папку с файлами программы необходимо вручную поместить файл нейросетевой модели qwen2.5-3b-instruct-q4_k_m.gguf, который можно скачать отдельно по предоставленной ссылке на Google Диск.
6. ОБЯЗАТЕЛЬНО установите на устройство "официальный пакет Microsoft (x64)" по следующей ссылке https://learn.microsoft.com/en-us/cpp/windows/latest-supported-vc-redist?view=msvc-170

Запуск приложения: Старт системы осуществляется через запуск основного файла app.py или через запуск файла "Запуск Системы.bat"
Дополнительная информация: Подробный принцип работы интеллектуальных агентов, этапы оцифровки и видеоматериалы с демонстрацией функций представлены в основном руководстве пользователя.

АЛЬТЕРНАТИВНЫЙ СПОСОБ: В папке на Google Диске представлен готовый zip-архив. Вы можете просто распаковать его и пользоваться приложением без ручной настройки, но установкой библеотек в файле requirements.txt
